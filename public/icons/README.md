# PWA Icons

Generate and place these PNGs here before deploying:

- `icon-192.png` — 192×192, any purpose
- `icon-512.png` — 512×512, any purpose
- `icon-maskable-512.png` — 512×512, **maskable** (10% safe padding inside the canvas — design fits inside a 410×410 circle)

Use https://maskable.app/editor or generate from Figma. Until real icons exist, browsers fall back to favicon.
