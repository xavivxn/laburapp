# laburapp

PWA tipo «Tinder» para laburos en LATAM: roles persona buscando laburo / empleador. **Next.js 16**, TypeScript y **Tailwind v4**. Backend Supabase cuando se cablee; hasta entonces datos mock.

## Desarrollo

```bash
npm install
npm run dev
```

Abrí [http://localhost:3000](http://localhost:3000).

Plantilla de variables: copiá `.env.local.example` a `.env.local` (no subas `.env.local` al repo).

## Deploy

[Documentación oficial de Next.js — deploy](https://nextjs.org/docs/app/building-your-application/deploying) (ej. **Vercel**).
